MAlice Specification

Simple Structures
UCHARACTER 	-> 'A' | 'B' | .. | 'Y' | 'Z'
LCHARACTER 	-> 'a' | 'b' | .. | 'y' | 'z'
CHARACTER		:: = UCHARACTER | LCHARACTER | '_'
DIGIT			-> 0 | 1 | .. | 8 | 9
NUMBER 		-> DIGIT | DIGIT NUMBER
ALPHANUMERIC	- CHARACTER | DIGIT
STRING		-> CHARACTER | CHARACTER STRING
ALPHASTRING 	-> ALPHANUMERIC | ALPHANUMERIC ALPHASTRING
SEPERATOR 	-> ',' | '.' | 'and' | 'but' | 'too'
TERMINATOR	-> '.' | 'too.

Alice Structures
TYPE			-> 'letter' | 'NUMBER'
OBRACE		-> 'opened'
CBRACE		-> 'closed'
IDENT			-> STRING
EXP			-> EXP '+' TERM | EXP '-' TERM | TERM
TERM			-> TERM '*' FACTOR | TERM '/' FACTOR | FACTOR
FACTOR		-> NUMBER | IDENT
DECLARE		-> IDENT 'was a' TYPE
ASSIGN		-> IDENT 'became' (EXP | UCHARACTER | LCHARACTER)
RETURN		-> EXP 'said Alice'
CODE			-> DECLARE | ASSIGN | RETURN
CODEBLOCK		-> CODE | CODE SEPERATOR CODEBLOCK | 
					CODE TERMINATOR
FUNCTION 		-> STRING '()' OBRACE CODEBLOCK CBRACE
